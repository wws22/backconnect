==================
README for backconnect.sh
==================

This is the small script to create back connect to your own host

Installing
==========

Install the newest version from github::

   git clone https://github.com/wws22/backconnect.git

Using
========

.. code-block:: bash

    $./backconnect.sh YOUR_IP
